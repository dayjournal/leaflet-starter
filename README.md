# leaflet-starter

![README02](img/README02.png)

![README05](img/README05.png)

Start Leaflet easily.  
- [Leaflet v1.7.1](http://leafletjs.com) 
- [webpack v5.38.1](https://webpack.js.org)  
- node v16.3.0
- npm v7.15.1

<br>

## Usage

![README04](img/README04.png)

<br>

Install package
```bash
npm install
```

<br>

build
```bash
npm run build
```

<br>

dev
```bash
npm run dev
```
